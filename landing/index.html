// how it works
The IKEv2
handshake.
Seven steps from your device to fully encrypted internet.
01

Enter credentials

Username, password, and server address entered in native OS VPN settings.

02

IKE_SA_INIT

Client initiates cryptographic handshake with the gateway on UDP 500/4500.

03

Server responds

Gateway sends encryption parameters and X.509 certificate for verification.

04

EAP-MSCHAPv2 auth

Credentials authenticated. Session keys derived via Diffie-Hellman key exchange.

05

IPsec tunnel established

ESP encrypts all packets at the network layer with AES-256-GCM.

06

Virtual IP assigned

StrongSwan allocates a virtual IP from the 10.10.10.0/24 subnet.

07

NAT + full routing

All traffic exits through the server's public IP. Your real IP is hidden.

// faq
Common questions.
Do I need to install an app?+
No. Turnip VPN uses the IKEv2 protocol built into iOS, macOS, Windows, and Linux. On iOS and macOS, open the .mobileconfig file you receive and the VPN is configured in one tap. On Android, use the free strongSwan app.
How quickly is my account activated?+
Instantly. As soon as your Paystack payment is confirmed, our system automatically creates your VPN credentials and emails you the setup profile. The whole process takes under 30 seconds.
Do you keep logs of my activity?+
No. We log connection timestamps and server performance metrics only. Your DNS queries, browsing history, and traffic contents are never stored. We have nothing to hand over because we never collect it.
What encryption does it use?+
AES-256-GCM for data encryption, SHA2-256 for integrity verification, and Diffie-Hellman Group 14 (MODP-2048) for key exchange. IKEv2 provides perfect forward secrecy — each session uses unique keys that cannot decrypt past or future sessions.
Can I use it on multiple devices?+
Yes. Basic supports 1 device, Pro supports 5 simultaneous devices, and Business supports unlimited devices — all using the same credentials. Each device gets its own setup guide in your customer dashboard.
What happens when my subscription expires?+
You'll receive a renewal reminder 3 days before expiry. After expiry, your VPN account is automatically deactivated. Renew anytime from your dashboard and new credentials are issued instantly.
